ToC path:

mcorest_api_json_write_*

For an overview see page Web Services

The following table lists the functions used to write JSON data to the web service caller. Please use the links to view detailed descriptions and examples:

mcorest_api_json_write_start() Start writing JSON to a request's response stream
mcorest_api_json_write_finish() Finish writing JSON data to the stream
mcorest_api_json_write_key() Write a JSON key
mcorest_api_json_write_object_begin() Start a JSON object
mcorest_api_json_write_object_end() End a JSON object
mcorest_api_json_write_array_begin() Start a JSON array
mcorest_api_json_write_array_end() End a JSON array
mcorest_api_json_write_bool() Write a Boolean value
mcorest_api_json_write_long() Write a long integer value
mcorest_api_json_write_ulong() Write an unsigned long integer value
mcorest_api_json_write_i64() Write a 64-bit integer value
mcorest_api_json_write_u64() Write a 64-bit unsigned integer value
mcorest_api_json_write_double() Write a floating-point value
mcorest_api_json_write_string_begin() Start writing a string value
mcorest_api_json_write_string_part() Write a fragment of a string value
mcorest_api_json_write_string_end() End the string value
mcorest_api_json_write_string() Write a string
mcorest_api_json_write_null() Write a null
mcorest_api_json_write_hex_string_part() Write a fragment of a hexadecimal representation of binary data
mcorest_api_json_write_hex_string() Write binary data converted to hexadecimal string representation
